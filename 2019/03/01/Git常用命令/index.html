版本回退

  • HEAD指向的版本就是当前版本，因此Git允许我们重新指定所有已提交的版本中的一个，使用命令git reset –hard commit_id。

  • 想要回退到哪个版本，可用git log查看提交历史。

  • 想要回到未来的哪个版本来，可用git reflog查看命令历史。
